# Basic Authentication API

Login and Register system with NodeJS

## Getting Started

### Usage & Development

1. **Clone repository**
```
git clone https://github.com/aftekardic/nodejs_authentication_system.git
```

2. **Install dependencies**
```
npm install
```

3. **Run API**
```
npm start
```


## Technologies Used
###

- Node.js
- MongoDB
